The author discovered that the underlying of Github Copilot Chat is calling the ChatGPT interface, so this tool was created.

It can convert ChatGPT requests into Github Copilot Chat requests. As long as you have a Github Copilot account, you can use the GPT-4 model of ChatGPT without any restrictions.
