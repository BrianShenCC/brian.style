https://leetcode.com/problems/check-if-all-as-appears-before-all-bs/

> Solution

``` typescript
function checkString(s: string): boolean {
    const lasta = s.lastIndexOf('a');
    const firstb = s.indexOf('b');
    return firstb > lasta || firstb === -1;
};
```
