> Problem
https://leetcode.com/problems/keep-multiplying-found-values-by-two/

> Solution

``` typescript
function findFinalValue(nums: number[], original: number): number {
    let temp = original;
    while (nums.includes(temp)) {
        temp = temp * 2;
    }
    return temp;
};
```
